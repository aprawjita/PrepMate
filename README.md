
# 🧠 PrepMate

**PrepMate** is an AI-powered educational platform that transforms static university syllabi into dynamic, gamified mock exams. Built with a robust Spring Boot backend and the Google Gemini AI API, it allows students to generate infinite practice tests tailored exactly to their coursework.

## ✨ Key Features
* **Knowledge Base Sync:** Upload a course syllabus PDF. The system extracts text, uses AI to parse the curriculum into structured JSON, and persists the units/topics into a PostgreSQL database.
* **Advanced Exam Generation:** The AI randomly selects 5 distinct topics from the database to create a comprehensive mock exam featuring multiple-choice and 7-mark descriptive questions.
* **PYQ Intelligence:** Optionally upload Previous Year Question (PYQ) papers to teach the AI the specific difficulty, tone, and formatting of your university exams.
* **Gamified "Dark Mode" UI:** A sleek, interactive frontend featuring an XP scoring system, answer streak multipliers, dynamic progress bars, and haptic visual feedback.
* **Intelligent Feedback:** Every question includes a "Professor's Note" or "Grading Rubric" generated by the AI to explain the correct answers.

## 🛠️ Tech Stack
* **Backend:** Java 17, Spring Boot, Spring Data JPA
* **Database:** PostgreSQL
* **AI Integration:** Google Gemini 2.5 API (Strict JSON Mode)
* **Frontend:** HTML5, CSS3 (Custom Dark Theme), Vanilla JavaScript
* **Tools:** Maven, Apache PDFBox
